The studio — palette workbench

Retune the MAP's whole colour system by eye. The two real levers are here: the category colours (they drive pills, graph nodes, explorer dots, the note-type rail, link hovers) and the light / dark base palette. Edit a colour and the live mocks on the right update instantly. When you like it, copy the export blocks at the bottom and paste them in. A sandbox — these values are TS/SCSS, so a change goes live only after npx quartz build.

Category colours

One colour per note type. Duplicated across five files (graph, categoryNav, NoteTypeBar, noteType, explorer) — the export keeps them in sync.
